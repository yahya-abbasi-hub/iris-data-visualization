#  Iris Data Visualization

##  Project Overview
This project performs Exploratory Data Analysis (EDA) on the famous Iris dataset using Python.

The goal is to understand the data through visualizations and identify patterns, relationships, and outliers.

---

##  Tasks Performed
- Loaded dataset using seaborn
- Inspected data using pandas (.head(), .info(), .describe())
- Created scatter plots to analyze feature relationships
- Built histograms to study data distribution
- Used box plots to detect outliers
- Generated pair plot for overall visualization

---

##  Dataset Information
- Total Samples: 150
- Features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- Target:
  - Species (Setosa, Versicolor, Virginica)

---

##  Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

##  Visualizations
The project includes:
- Scatter Plots
- Histograms
- Box Plots
- Pair Plot

(All plots are saved as PNG files in the repository)

---

##  How to Run
1. Open the notebook in Google Colab or Jupyter Notebook
2. Run all cells step by step
3. View generated plots

---

##  Key Insights
- Petal features show clear separation between species
- Sepal features have some overlap
- Some outliers detected in sepal width
